* When a character is sent to the output
* what key sequence does it represent. 
* 
* IN_MAP_<ascii>=<byte-sequence|ansi-sequence-string|unicode-value>
*
* byte-sequence - Comma delimited bytes. If you put '$' on the byte
*                 it will be read as hex.
*
* ansi-sequence - If value starts with '^' it will be read as an
*                 ANSI sequence, with '^' being the escape
*
* unicode-value - If a value starts with 'U' or 'u', it will be
*                 read as a unicode value. The value must be specified
*                 as hex.
********************************
